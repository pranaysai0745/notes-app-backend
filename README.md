# Notes App Backend API

A professional Notes Backend API built using FastAPI, JWT Authentication, and SQLite/PostgreSQL.

---

# Features

- User Registration
- User Login with JWT Authentication
- Create Notes
- Get All Notes
- Get Single Note
- Update Note
- Delete Note
- Share Note
- Search Notes
- Protected Routes using JWT
- SQLAlchemy Database Integration

---

# Tech Stack

- FastAPI
- Python
- SQLAlchemy
- JWT Authentication
- SQLite
- Uvicorn
